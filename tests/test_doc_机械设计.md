# 机械设计基础 — 齿轮传动与轴承

## 齿轮失效形式

齿轮是机械传动中最常用的零件之一。齿轮的主要失效形式包括：

1. **齿面磨损** — 由于啮合齿面间存在相对滑动，在载荷作用下产生磨粒磨损或粘着磨损。主要发生在开式齿轮传动中。
2. **齿面点蚀** — 在循环接触应力作用下，齿面表层产生微小疲劳裂纹，裂纹扩展导致小片金属剥落。常见于闭式软齿面齿轮。
3. **齿面胶合** — 在高速重载条件下，齿面温度升高，油膜破裂，金属直接接触发生粘着撕裂。
4. **轮齿折断** — 包括疲劳折断和过载折断，是最危险的失效形式，通常导致传动立即中断。
5. **塑性变形** — 齿面材料在过大应力下发生塑性流动，导致齿形改变。

预防措施：合理选择材料和热处理、保证润滑、控制载荷和速度。

## 齿轮材料选择

### 常用齿轮材料

| 材料 | 热处理 | 硬度(HB) | 应用场景 |
|------|--------|----------|----------|
| 45钢 | 调质 | 217-255 | 一般用途齿轮 |
| 40Cr | 调质 | 241-286 | 中等载荷齿轮 |
| 20CrMnTi | 渗碳淬火 | 56-62(HRC) | 高速重载齿轮 |
| 38CrMoAl | 氮化 | >850(HV) | 高精度耐磨齿轮 |
| QT600-3 | 正火 | 197-269 | 中低速大齿轮 |

### 选材原则
- 软齿面齿轮 (HB≤350)：适用于中低速、一般精度要求的场合，制造简单
- 硬齿面齿轮 (HB>350)：承载能力高，寿命长，但制造成本较高
- 配对齿轮的硬度差应在 30-50HB 之间

## 滚动轴承选型

滚动轴承是机械中用来支承轴及轴上零件的部件。选型需考虑以下因素：

1. **载荷大小与方向** — 径向载荷大选深沟球轴承或圆柱滚子轴承；轴向载荷大选推力球轴承；联合载荷选角接触球轴承或圆锥滚子轴承。
2. **转速** — 球轴承的极限转速一般高于滚子轴承。高速场合宜选轻系列或特轻系列轴承。
3. **调心性能** — 当轴的刚度不足或轴承座孔同轴度较差时，应选调心球轴承或调心滚子轴承。
4. **安装与拆卸** — 需要经常装拆的场合选内圈带锥孔的轴承或分离型轴承。

### 轴承代号说明

例：**6208** — 6: 深沟球轴承, 2: 轻系列, 08: 内径 40mm

## 机械设计基本准则

机械零件设计应满足以下基本要求：
- **强度准则**：零件在工作时不发生断裂或残余变形
- **刚度准则**：弹性变形不超过允许值
- **寿命准则**：在规定的使用期限内不发生疲劳破坏
- **可靠性准则**：在规定条件下和规定时间内完成规定功能的能力
- **经济性准则**：在满足功能要求的前提下成本最低
